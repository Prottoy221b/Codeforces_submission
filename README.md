# Codeforces Submissions

This repository contains my solutions to problems from Codeforces, organized by difficulty.

## 📁 Structure

Problems are grouped by rating:

```
800/
900/
1000/
```

Each file is named using:

```
<problemID>_<problemName>.cpp
```

Example:

```
800/4A_Watermelon.cpp
```

## 🧠 Approach

* Focus on problem-solving and logic building
* Each solution includes comments explaining:

  * Approach
  * Key idea
  * Time complexity
